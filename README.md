## Getting Started

To get a local copy up and running follow these simple steps:

1. Clone the repo
   ```sh
   git clone https://github.com/Spafi/todo-frontend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```
   npm start
   ```
